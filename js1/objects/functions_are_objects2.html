Now let's take a look at a couple methods that are available in your functions. Here we have this call method, and with this we can call a function. Look at the arguments. When you hover over the argument editor shows the first argument as this argument . Here we need to pass an empty object{} and this keyword in the constructor function will reference this empty object that we pass in call. After that we have to pass other parameters the function needs.

Note
hi.call({},"Nithin");
let msg1 = new hi("Nithin"); when we use the new operator , this new operator will internally create an empty object and pass that as the first argument to the call method.

If you don't use new keyword while using constructor function, then this keyword in constructor function will point to global object windows. So if you want to do this in call method, instead of sending empty object{} as first parameter, you needs to send windows.
hi.call(window,"Nithin");
let msg1 = hi("Nithin");

Now we have another method called apply. It's exactly like the call method, with this we can call a function, but instead of passing all the arguments explicitly we pass them in an array. So this is useful if you already have an array somewhere in your application, and you want to pass an array as the second argument to the apply method.

Property constructor references the function that was used to construct or create that object. Inspect this webpage.

When we look at constructor property of factory functions we get f which is function and Object(). What is that Object()? This is a built in constructor function in JavaScript.

When we create an object using object literal syntax => {}(which we use for factory functions), internally the JavaScript engine uses this constructor function For example, when we create an object using object literal syntax
let x ={};
js will do something like this
let x = new Object();
where Object() is built-in fuction. So our object Student1 which was created by factory functions, is created using constructor function Object() internally.

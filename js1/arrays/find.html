Finding elements in an array

Finding elements depend whether you are storing primitive or reference type. Let's start with primitive.
Let array be : const arr = [1,'a',1,null,true,];

indexOf()

We use method indexOf to check whether element is present in the array. If yes it returns index of the element. Else it returns -1.

console.log(arr.indexOf('a')); //returns 1
console.log(arr.indexOf(1)); //returns 0
console.log(arr.indexOf(true)); //returns 4
console.log(arr.indexOf(null)); //returns 2
console.log(arr.indexOf(undefined)); //returns 5
console.log(arr.indexOf(4)); //returns -1

Note type of argument you are passing always matters. If you pass '1' as argument (arr.indexOf('1')), it considers 1 as string and returns -1 as element 1 of type string is not present.

includes()

Similarly there is another method called includes that returns true if the element is present in the array.
console.log(arr.includes(undefined));
console.log(arr.includes(4));

lastIndexOf()

Element 1 is present twice in our array. When we use indexOf() it returns first occurence. To get last occurrence.
console.log(arr.lastIndexOf(1));

Now, all thee methods have a second parameter which is optional. And that is the starting index. This is the index where operation begins. For example,
arr.indexOf('a',2)
starts searching for element 'a' from 2nd index.

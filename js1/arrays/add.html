Let's define an array: const arr = [3,6];

Note we used const to declare an array. So, we cannot reassign an array but we can modify them. Which means:
arr = [1,2,3,4];//throws an err

We know that arrays are objects. We use methods to add elements in array.
To add element at the end of an array: arr.push(7,8);
To add element at the beginning of an array: arr.unshift(1,2);
To add element at the middle of an array: arr.splice(3,0,4,5);

To add element middle of an array we specify the index at which we should add element so we have passed 3 as our first argument. We passes 0 as as our second argument. We will talk about this in the removing elements lesson but for now keep in mind that if you want to insert elements without deleting anything set zero as second parameter.

There are three different ways to include css to your html doc

  • embedded stylesheet
  • external stylesheet
  • inline stylesheet

Embedded stylesheet is nothing but including style sheet in head section with <style> tag. example:

<style>
p {
color: orange;
}
</style>

external style sheet is nothing but including all those styles in a different file and saving it with .css extension. After that you need to include it with <link> tag.

Example: <link rel="stylesheet" href="style.css">

inline style sheet is nothing but including style within html tag.

Example: <p style = "color = blue; font-weight = bold ">
Note if the html document has external and embedded stylesheet which have different property for same tag then external style sheet is overwritten and embedded style is followed. If html doc also have inline then html follows inline styles.

Also keep in mind that different browsers have different default rendering properties which means some browsers may have 0 margin whereas some may have 10px as margin. So to avoid this, we should overwrite all those default rendering. To save time there is a website called normalize css which have done this work for you. You can copy that and create a new stylesheet and include ut in your html document.

In this webpage we are going to see about hyperlinks. Always webpages have links that navigates to same page or different page. We use anchor element to create these links.


href is short for hyper text reference which means url.

Lets make a image as link

when you hover over an image icon changes to hand indicating that image is clickable

relative url search for webpage with respect to folder where this document is.
For example let us assume root folder has two folders named html_css_1 and js1 where folder 1 has lesson1.html and lesson6.html whereas folder2 has index.html to goto lesson1 webpage which is present in same folder,
<a href ='lesson1.html'>
to goto index webpage which is present in different folder,
<a href ='../js1/index.html'>

If you are in deeply nested structure you want to go multiple levels up . For example consider you are present 4 folders inside from home directory and you want to go to index page which is in home directory. Then relative url looks like ../../../../js1/index1.html which goes 4 levels up.But this looks messy so instead we use absolute url <a href ='/index.html'> where / represents home directory.



You can aslo link to non-html docs like images, pdfs etc. <a href ="pic/myimg.jpg">
If you want to make user download the image then add download attribute <a href ="pic/myimg.jpg" download>

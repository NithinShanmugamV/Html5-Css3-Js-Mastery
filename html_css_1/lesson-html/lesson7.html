Unsplash website has lot of free images. So you can search and use images those images.
While naming the image, you can name whatever you want but always give a descriptive name of what is in the image. This is a search Engine Optimization trick, by which search engines can understand webpages better and show your results at top.

Although alt is not a necessary attribute, it can be useful for following reasons:
  • When visually impaired people uses screen reader it reads the alt text for them when it encounters with image.
  • It is also used in Search Engine Optimization.
  • It is also used when image doesn't load. When image doesn't load it shows the alt text.
